Swift开发macOS应用01


2020-10-01
JAVA基础
作者: 双king
浏览量:次

这个系列主要是翻译来自苹果官方网站的使用swift开发macOS应用，网上资料比较少，相信看到这里的人，都不需要再解释一些例如xcode、macOS、macOS Big Sur等基本概念了，这里的各个软件的版本如下：

  • xcode 12
  • macOS Big Sur

1、新建项目

1、为了能实时预览效果，你需要升级系统到最新的macOS Big Sur，在xcode12中，选择 File > New > Target. 选择macOS标签下的App，选择next。

2、填Product Name等，看截图：

3、设置 scheme, MacLandmarks > My Mac.这样才能预览，在本教程中，您将使用其他方案来监视其他目标如何响应共享文件中的更改。

4、选择ContentView.swift，打开Canvas，点击Resume按钮来看一下预览，注意SwiftUI可以像你在IOS中那样，预览应用的主窗口。

5、回到项目的导航栏，在MacLandmarks group中，删除MacLandmarksApp.swift，点确认。你也不需要默认的结构，一般你都会自己搞一个。

接下来，你会和IOS App共享view model 和 resource 文件

6、在项目导航栏，按住command键，选择以下文件，LandmarksApp.swift,LandmarkList.swift, LandmarkRow.swift, CircleImage.swift, MapView.swift, and FavoriteButton.swift`.第一个文件LandmarksApp.swift是共享App定义的地方，其它的都是在macOS上生效的。

7、按住command继续选择其它的一些文件，这些文件定义了应用的data model和resources （这个就不翻译了，英文可能会更好理解，就是数据模型和资源文件）

8、 在文件检查器中，将MacLandmarks添加到选定文件的目标成员资格中。 还需要选择1个icon图标

9、选择Assets.xcasset文件，删除空的一些AppIcon内容，接下来你要替换这些内容
